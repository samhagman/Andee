# Skill Authoring Best Practices

This skill provides comprehensive guidance for writing effective Claude Code skills.

## What's Covered

The full reference guide contains the following sections:

- **Core principles** - Conciseness, setting appropriate degrees of freedom, testing with different models
- **Skill structure** - YAML frontmatter, naming conventions, writing effective descriptions, progressive disclosure patterns
- **Workflows and feedback loops** - Breaking complex tasks into steps, implementing validation loops
- **Content guidelines** - Avoiding time-sensitive info, using consistent terminology
- **Common patterns** - Template pattern, examples pattern, conditional workflow pattern
- **Evaluation and iteration** - Building evaluations first, iterative development with Claude, observing skill usage
- **Anti-patterns to avoid** - Windows-style paths, offering too many options
- **Advanced: Skills with executable code** - Error handling, utility scripts, visual analysis, verifiable outputs, package dependencies, runtime environment, MCP tool references
- **Technical notes** - YAML frontmatter requirements, token budgets
- **Checklist for effective skills** - Core quality checklist, code and scripts checklist
